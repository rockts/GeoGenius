# GeoGenius — 小学几何公式动画推导

小学数学（1-6年级）几何可视化学习工具，通过 SVG 逐步动画推导公式，内置 AI 讲解和随机练习题。

## 快速开始

直接用浏览器打开 `geometry_tool.html`，无需安装任何依赖。

## 功能

- **分步动画推导**：每个知识点拆分为 3-5 步，用 SVG 动画直观演示公式来源
- **100+ 动画步骤**：自定义缓动引擎，`requestAnimationFrame` 驱动，自动适配视口
- **AI 老师对话**：配置 DeepSeek API Key 启用真实 AI 讲解；不配置则使用本地 mock 回复
- **随机练习题**：19 个知识点各含一道选择题，含提示和即时反馈
- **学习进度记录**：已访问的卡片显示 ✓ 标记，年级选择持久化
- **键盘快捷键**：→/Space 下一步，← 上一步，R 重播，Esc 返回

## 知识点覆盖

依据义务教育数学课程标准（2022年版），覆盖「图形认识与测量」和「图形位置与运动」两大主题：

| 年级 | 图形认识与测量 | 图形位置与运动 |
|------|--------------|--------------|
| 一年级 | 认识立体图形、认识平面图形 | — |
| 二年级 | 角的初步认识、长方形与正方形 | 轴对称图形、平移与旋转初步 |
| 三年级 | 四边形与平行四边形、周长的认识、长方形周长公式、面积的认识、长方形面积公式 | — |
| 四年级 | 角的度量、平行四边形与梯形 | 轴对称深化、平移深化、旋转深化 |
| 五年级 | 平行四边形面积、三角形面积、梯形面积、组合图形面积 | 图形的放大与缩小 |
| 六年级 | 圆的周长、圆的面积、圆柱表面积、圆柱体积、圆锥体积 | — |

**合计：28 个知识点，100+ 动画步骤**

## AI 设置

点击右上角「AI设置」，输入 [DeepSeek API Key](https://platform.deepseek.com/)（注册后免费获取）。Key 仅存储在本地 `localStorage`，不会上传服务器。

不填写则使用本地 mock 回复，所有功能正常可用。

## 技术实现

单文件 HTML，无外部框架，无构建步骤：

| 类别 | 实现 |
|------|------|
| 图形渲染 | 原生 SVG + 自封装辅助函数（E/T/R/PL/L/C/RA） |
| 动画引擎 | `requestAnimationFrame` + ease-in-out 缓动 |
| 布局 | CSS Grid + Flexbox，响应式适配移动端 |
| 持久化 | `localStorage`（年级、访问记录、API Key） |
| AI 接入 | DeepSeek Chat API（OpenAI 兼容格式） |
| 字体/图标 | Noto Sans SC + Tabler Icons（CDN） |

## 已知限制与后续计划

- 练习题目前每个知识点仅有 1 道（19 个知识点有题，9 个运动主题暂无）
- AI mock 回复较简单，配置真实 Key 可获得更好的讲解效果
- 所有代码内联在单个 HTML 文件中，如需持续开发建议迁移至模块化结构
